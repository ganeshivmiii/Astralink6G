# Astralink6G

Premium responsive AI dashboard for 6G network intelligence.

## Features

- Responsive Streamlit UI for desktop and mobile
- Premium red/white/black telecom-style interface
- Interactive congestion prediction with scenario presets
- Live risk index and loading animation
- Network analytics and selectable telemetry metrics
- Risk insights table
- Smart routing visualization

## Run Locally

```bash
pip install -r requirements.txt
streamlit run src/dashboard.py
```

## Streamlit Cloud

Deploy from GitHub with:

- Main file path: `src/dashboard.py`
- Python dependencies: `requirements.txt`
